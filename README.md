# Catalyst Analytics Website

A modern and responsive landing page for **Catalyst Analytics**, built with **HTML** and **Tailwind CSS**.
This website showcases digital solutions, services, and FAQs with a sleek dark UI and engaging visuals.

---

## Preview

The website includes:

* Modern Navbar
* Hero Section
* Client Logos Section
* Services Section
* FAQ Section
* Footer

---

## Features

* Fully responsive design
* Modern dark-themed UI
* Built with Tailwind CSS
* Clean and organized layout
* Interactive FAQ section
* Smooth and user-friendly interface

---

## Technologies Used

* HTML5
* Tailwind CSS
* Git & GitHub
